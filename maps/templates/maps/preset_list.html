About Presets

What are presets?

Presets allow you to save your favorite map generation settings so you can quickly create similar maps without re-entering all the parameters each time.

Public vs Private:

  • Private: Only you can see and use your preset
  • Public: Other users can see and use your preset (but cannot edit it)
